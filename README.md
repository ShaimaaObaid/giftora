# Giftora

Giftora is a smart gift recommendation platform built with Django.

The platform helps users discover suitable gifts based on occasion, budget, and recipient type, making gift selection faster and more personalized.

## Features

* User Registration and Login
* Gift Suggestions
* Dynamic Gift Search
* AJAX Integration
* Admin Dashboard
* Responsive Design

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* AJAX
* SQLite

## Future Improvements

* AI Gift Assistant
* MySQL Integration
* AWS Deployment
* Advanced Personalization
